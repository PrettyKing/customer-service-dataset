# 演示商城售后政策

> 本文档仅用于本地客服系统功能演示，不代表真实商家的正式政策。

## 七天无理由退货

商品自签收次日起七天内，在商品完好、配件齐全且不影响二次销售的情况下，可以申请七天无理由退货。定制商品、已拆封的贴身用品和法律规定不适用无理由退货的商品除外。

## 质量问题

商品存在质量问题时，用户应提供订单号、问题照片或视频。客服核实后可按规则安排退货、换货或维修。客服不能在未核实前承诺赔偿金额。

## 退款到账

退货商品验收通过后，退款会原路退回。支付机构处理通常需要一至五个工作日，具体到账时间以银行或支付平台为准。

## 人工升级

涉及人身安全、重大财产损失、重复投诉、平台介入或用户明确要求人工处理时，应停止自行承诺并转交人工客服。
